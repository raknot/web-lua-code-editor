## Features

- **Multi-file Support** - Work with multiple Lua files simultaneously
- **Syntax Highlighting** - Monaco Editor with full Lua language support
- **Error Checking** - Real-time syntax validation with error highlighting
- **Code Formatting** - Auto-indent and format your code
- **Export Files** - Download your work as .lua files
- **Mobile Ready** - Fully responsive design for coding on the go

## Quick Start

Click the banner above to try the editor, or visit [raknot.dev/example/editor](https://raknot.dev/example/editor)

## Usage

1. **Create files** - Use the `+ Add File` button to create or import Lua files
2. **Write code** - Use the Monaco editor with IntelliSense and auto-completion
3. **Check syntax** - Click `Check Syntax` to validate your code
4. **Format code** - Click `Format` to auto-indent properly
5. **Export** - Click `Export` to download as .lua file

## Keyboard Shortcuts

- `Ctrl + /` - Toggle comment
- `Ctrl + F` - Find
- `Ctrl + Z` / `Ctrl + Y` - Undo / Redo
- `Ctrl + D` - Duplicate line

## Tech Stack

Built with Monaco Editor (VS Code engine), Lua Parser, and vanilla JavaScript. No backend required, runs entirely in your browser.
