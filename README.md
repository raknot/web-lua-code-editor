## Features

- **Multi-file Support** - Work with multiple Lua files simultaneously
- **Syntax Highlighting** - Monaco Editor with full Lua language support
- **Error Checking** - Real-time syntax validation with error highlighting
- **Code Formatting** - Auto-indent and format your code
- **Export Files** - Download your work as .lua files
- **Mobile Ready** - Fully responsive design for coding on the go
- **Copy Code** - Quick copy button to save code to clipboard
- **Code Snippets** - Insert common Lua patterns (loops, functions, tables, etc.)
- **File Management** - Right-click tabs to rename files, duplicate files with one click
- **No Memory** - Privacy-focused: files are never saved between sessions

## Quick Start

Click the banner above to try the editor, or visit [raknot.dev/example/editor](https://raknot.dev/example/editor)

## Usage

1. **Create files** - Use the `+ Add File` button to create or import Lua files
2. **Duplicate files** - Click `+ Add File` and select `Duplicate Current File` to clone the active file
3. **Rename files** - Right-click any tab to rename the file
4. **Write code** - Use the Monaco editor with IntelliSense and auto-completion
5. **Insert snippets** - Click `Snippets` to insert common Lua code patterns
6. **Check syntax** - Click `Check Syntax` to validate your code
7. **Format code** - Click `Format` to auto-indent properly
8. **Copy code** - Click `Copy Code` to save to clipboard
9. **Export** - Click `Export` to download as .lua file

## Keyboard Shortcuts

- `Ctrl + /` - Toggle comment
- `Ctrl + F` - Find
- `Ctrl + Z` / `Ctrl + Y` - Undo / Redo
- `Ctrl + D` - Duplicate line

## Tech Stack

Built with Monaco Editor (VS Code engine), Lua Parser, and vanilla JavaScript. No backend required, runs entirely in your browser.
